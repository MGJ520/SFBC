# FOC-Code

## 介绍
学习Foc控制，PID算法，项目Fork from YeHarold/BalanceFocCar

## 项目简介
基于ESP32S3芯片和Arduino框架下Simplefoc库开发的无刷电机平衡车

## 硬件
#### 电机：ZD2808
#### 主控：ESP32S3-WROOM-U1-N16R8
#### MPU:MPU6050（I2C）
#### 编码器：AS5600（I2C）
#### 无刷电机驱动：DRV8300DRGER
#### 低端电流采样：INA181A2IDBVR

## 快速入门
#### 软件平台:PlatformIO
#### 嵌入式框架:Arduino
